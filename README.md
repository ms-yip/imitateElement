### 步骤

- 环境：vue-cli3         node 8.10+
- 打开项目 进入项目目录
- 执行 npm install
- 执行 vue add element 
- 然后 npm run serve

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
